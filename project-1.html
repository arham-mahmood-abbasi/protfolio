Project Overview

As the main focus Robo promo project builds an AI-driven marketing automation system which enhances content development service alongside campaign control functions along with monitoring capabilities. Generative AI with Robotic Process Automation serves as a foundation to decrease manual work while enhancing personalized marketing and general marketing achievement.

I developed a comprehensive full-stack web application and AI-powered content generation system, featuring a robust Django-based backend with RESTful APIs, BERT-driven keyword extraction, and Stable Diffusion for image generation, supplemented by real-time Google image scraping. The system automates PDF generation using Puppeteer, email delivery via SendGrid, and voice synthesis from user prompts, while integrating with a SvelteKit front-end for a responsive UI and Vue.js for real-time content previews. Security was enhanced with CAPTCHA verification, and website performance was optimized through lazy loading and code splitting. Additionally, I implemented a meta-campaign management system powered by LLaMA 3, leveraging self-supervised learning to analyze target audiences and generate personalized marketing campaigns, continuously improving based on performance insights.
